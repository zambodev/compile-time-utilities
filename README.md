# Compile Time Utilities
A set of C++ templatic utilities for an easier exprerience with various compile time structures

## Utilities
- Compile time array (obviously)
    - Array utilities
- Compile time string
- Compile time pair
- Compile time map


## Compile time map
The idea of a compile time map is what created all the other utilities
'ctmap' guarantee O(1) access time and a really small memory overhead
